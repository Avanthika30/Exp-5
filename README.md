# Exp - 5 STUDENT ENROLLMENT SYSTEM

# AIM:
To design a Student Enrollment System that allows students to register for courses and exams online, with mentor guidance and admin approval.
# SRS (Procedure):
1. Login: Student logs into the system.
2. Course Selection: Student views available courses with mentor reviews and advice.
3. Eligibility Check: System checks prerequisites, time conflicts, and maximum credit limits.
4. Confirm Registration: Student confirms selected courses.
5. Exam Selection: Student selects exams; system checks eligibility and timetable conflicts.
6. Approval: Admin reviews and approves course registration and exam selection.
7. Confirmation Slip: System generates a confirmation slip for the student.
